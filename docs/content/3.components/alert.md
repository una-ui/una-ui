| ---------- | --------------------------------------------------------------------------------- |

## Slots

| Name          | Props | Description                                |
| ------------- | ----- | ------------------------------------------ |
| `default`     | -     | Cover the `title` and `description` slots. |
| `title`       | -     | The title of the alert.                    |
| `description` | -     | The description of the alert.              |
| `icon`        | -     | The icon of the alert.                     |
| `closeIcon`   | -     | The close icon of the alert.               |
