### Variant

| Prop    | Default | Type        | Description               |
| ------- | ------- | ----------- | ------------------------- |
| `badge` | `soft`  | `{variant}` | The variant of the badge. |

| Variant   | Description                 |
| --------- | --------------------------- |
| `soft`    | The default variant.        |
| `solid`   | The solid variant.          |
| `outline` | The outline variant.        |
| `~`       | The unstyle or base variant |
